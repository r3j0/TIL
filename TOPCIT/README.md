- 최대한 많은 범위 끝내기.
- 중요한 것은 암기용에 따로 정리. 그 외에는 이해한 뒤 흐름을 적어놓거나 스킵할 것.
- 범위 공부도 하면서, 언어 공부도 정리하기.

- 기출문제를 봤는데, 실습 가능한 건 숙달해두기만 하고, 예시나 개념이 나오는 것은 키워드만 정리해서 암기하는게 좋을 듯
